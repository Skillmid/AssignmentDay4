Explain Mobile first design


Answer


Mobile first in Web Development is a concept in which designing of layout, navigation and content of a website is intially done for mobile views before designing the desktop versions for larger screens

This is good because it helps the designer in making good choices of layout and other needed part of the website to enhance proper positioning and view ports


Benefits of Using the Mobile-First Approach


It allows designers to focus not only on technicalities, but also on content, structure and navigation details.
It make Webpages easier to navigate, thus improving user experience.
Provides good site structure
Enables good user interface
