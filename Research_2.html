Explain progressive and adaptive design


Answer


Progressive design can be described as a web design strategy that makes an effort to create websites that are accessible to all devices, using a layered approach with a set of core principals stressing accessibility of content and functionality.


Layers of Progressive Designs


Content layer: This can be achived using HTML markup
Presentation layer: This is the implementation of CSS and Styling
Client-side scripting layer: This is the use of JavaScript or jQuery behavious

Adaptive design is useful for retrofitting an existing site in order to make it more mobile friendly. This allows you to take control of the design and develop for specific, multiple viewports.
